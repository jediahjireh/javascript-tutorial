JavaScript Tutorial: jQuery

jQuery is a fast, small, and feature-rich JavaScript library. It provides various functions for selecting elements, handling events and creating effects.

$(document).ready(function() {
    $('button').click(function() {
        // notify user of successful action
        alert("Button clicked!");

        // create element
        let newParagraph = $("<p></p>").text("Created paragraph.");
        // append element
        $("main").append(newParagraph);
            
        // effect chaining 
        $(this).css("color", "blue").slideDown(500).slideUp(500);
    });
});
